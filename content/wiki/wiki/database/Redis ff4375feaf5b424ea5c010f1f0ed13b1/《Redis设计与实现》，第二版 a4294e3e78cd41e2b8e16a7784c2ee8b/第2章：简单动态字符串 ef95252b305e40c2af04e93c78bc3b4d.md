# 第2章：简单动态字符串

- SDS：Simple Dynamic String
- 比如数据库中key都是用字符串表示的。
- SDS也可用作缓冲
- 定义文件：sds.h
- 结构体：struct sdshdr
    
    ```c
    struct sdshdr {
        unsigned int len;
        unsigned int free;
        char buf[];
    };
    ```
