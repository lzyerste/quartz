    git remote --prune origin

    Git 清理无效的远程追踪分支

    prune后删除本地分支

    远程分支被删后，本地跟踪的分支也可以被删。本地分支会显示gone字样。

    1

    git branch -vv | grep ': gone]'|  grep -v "\*" | awk '{ print $1; }' | xargs -r git branch -d

    Delete local GIT branches that were deleted on remote repository

    删除remote信息

    git branch -avv的时候会显示多个remote信息，虽然git remote -vv的时候已经不见了remote信息。

    可以直接将目录.git/refs/remote/移除掉。
